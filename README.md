# PartnerAI Cowork Marketplace

Plugin marketplace for [Claude Cowork](https://claude.ai) — productivity plugins for freelancers and small businesses.

## Available Plugins

| Plugin | Category | Description |
|--------|----------|-------------|
| [receipt-processor](https://github.com/PartnerAI-labs/receipt-processor) | Finance | Extract data from receipt photos, verify via web UI, upload to QuickBooks |

## Installation

Add this marketplace to your Claude Cowork instance, then install any plugin from the list above.

## Adding a Plugin

To list a new plugin in this marketplace:

1. Build your plugin following the [Claude Cowork plugin spec](https://github.com/PartnerAI-labs/cowork-marketplace)
2. Host it in a public GitHub repo
3. Open a PR adding your plugin entry to `.claude-plugin/marketplace.json`
